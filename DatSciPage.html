Least Different Group Matching Analysis & Visualization

Description:

A PySimpleGUI desktop application that analyzes an Excel file with individual data from cage mate pairs. While maintaining this cage mate pairing, the program tests every possible combination of data matches and tests the overall distance between each combination’s data points, resulting in the hypothetical group assignment combination whose data is the least different. An Excel file is created with tabs recording each step of analysis. These steps include:
  1. All possible group combinations
  2. The mean values of all possible group combinations
  3. The absolute difference between all possible group combinations
  4. The group assignments of the subjects that result in the least different mean values of all possible group combinations along with this mean difference value
Matplotlib is used to display a graph of the two lines that represent the least different values data, which are the lines whose points are as close together as possible given the data and cage mate pairing.

The program was created to solve a common problem of experimental design: given subject data from pair housed subjects, how does one assign these subjects to two groups (control vs. treatment) so that these group assignments are the least different from each other, thus avoiding any previous difference between groups biasing the results of the next experiment? As an additional constraint, the next experiment necessitates the assignment of these subjects to two groups while ensuring that cage mates are in the same group assignment (if one subject is stressed from its group’s treatment, this could affect the results of its yet-to-be tested partner). This program solves this problem by automating the process of finding the optimal group assignments.

Robinhood Stock Data Analysis & Visualization

Description:

A program that uses the Robin_Stocks python library to interface with the Robinhood API to gather/process personal Robinhood brokerage account information. Data is gathered and added to an updated CSV file. The Pandas library is used to create data frames to process and manipulate data. The information is used to measure stock portfolio performance and display this information graphically using the Matplotlib Python library. Assets are divided up into three categories: benchmark group, personal stock pick group, and cryptocurrency group. These categories are analyzed by % return, $ return and their percentage of entire portfolio. These are represented as a line graph over time, bar graphs with total current values and doughnut graphs as percent of portfolio values.

Script created using the Robin_Stocks Python library and the Robinhood API

  • Top line graph displaying dollar value change over time.
  • Line graph displaying percent change over time.
  • Bar graph displaying total change currently.
  • Pie chart displaying asset class's portfolio distribution.
  • Pie chart displaying each asset class's current return on investment.

Wheel Running Data Analysis & Visualization

Description:

This program uses PySimpleGUI to enable the user to compile and organize a tremendous amount of Excel file output data. The input file is an Excel sheet in which one column of data is present for each experimental subject. Each cell of the column represents one-minute of wheel running activity as wheel turn revolutions. Accordingly, each day contains 1440 entries per subject. The experiment can last several months, resulting in over a million data points to analyze over the course of the experiment. This program automates much of the data compilation and analysis by aggregating valid bouts of wheel running activity and computing essential running activity values. Program output is daily running value calculations compiled as a statistical test ready table. The GUI also has an option for the user to view a bar graph showing mean values of the subject's average wheel running behaviors during their active vs inactive light-cycle.

Activity Text File Data Extraction to Excel & Visualization

Description:

Through a PySimpleGUI interface, the user transfers behavior data that is initially output as a text file into a compiled Excel file. The user inputs a .txt file name and desired output file destination. This program automates the data collection and organization process. The Excel file output is structured so that the user can directly transfer the data into the labs preferred statistical testing program and immediately run the pertinent analysis. The program solves the problem of the initial .txt file output containing copious amounts of non-essential data and information in text format with the small amount of essential data being embedded in this heap of non-essential text. Before creating this program, the experimenter had to search through the text file to manually copy and paste necessary values into Excel cells one by one.
