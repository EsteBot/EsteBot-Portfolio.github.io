Mg/Kg Dose Calculator for Gram Body Weights

Description:

A Tkinter GUI desktop application. The application has three ways to calculate the correct compound weight to create Mg/Kg doses for systemic injections.
  1. The first method requires desired Mg/Kg dose and final solution volume to created. Outputs are dry compound weight in Mg and grams.
  2. The second method enables the user to input as many individual subject body weights in grams as needed. The program will total these weight inputs. Then the user inputs the desired Mg/Kg dose and final solution volume. The program will output dry compound weight in Mg and grams.
  3. Lastly, the user can input their desired Mg/Kg dose and total subject weight in grams. The calculator will output dry compound weight in Mg, grams, and the solution volume necessary.
There is also a button to see example calculations that demonstrate and explain all calculations used in the application for reference and educational purposes.

Behavioral Timer with Excel Data Output

Description:

A Tkinter desktop application for timing behaviors of experimental subjects. Up to three different behaviors can be quantified simultaneously along with the counting of another temporally discrete behavior. The timer enables the user to select the total time duration of the test trial. Total behavior bouts, total time of the behavior duration and the average time duration of each behavioral bout are output as an Excel file for further analysis.
